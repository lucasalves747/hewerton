# Hewerton Scheidegger - Páginas HTML Estáticas

Este projeto contém duas páginas HTML estáticas que podem ser executadas diretamente no navegador, sem necessidade de servidor.

## Arquivos

- `index.html` - Página principal com todas as seções (Sobre, Método REI, Livros, E-books, Depoimentos)
- `contact.html` - Página de contato com formulário

## Como usar

1. **Abrir no navegador**: Clique duplo nos arquivos HTML ou arraste-os para o navegador
2. **Navegação**: Use os links do menu para navegar entre as seções
3. **Formulários**: Os formulários usam Formspree para envio (configurado com endpoint público)

## Funcionalidades

### Página Principal (index.html)
- ✅ Navegação por scroll suave entre seções
- ✅ Menu responsivo para mobile
- ✅ Modal para download de e-books
- ✅ Formulário integrado ao Formspree
- ✅ Design responsivo
- ✅ Animações e efeitos visuais

### Página de Contato (contact.html)
- ✅ Formulário completo de contato
- ✅ Validação de campos
- ✅ Feedback visual de envio
- ✅ Navegação de volta para página principal

## Formulários

Os formulários estão configurados para usar o Formspree (serviço gratuito de formulários):
- **Endpoint**: `https://formspree.io/f/xpznqryq`
- **Campos**: Nome, telefone, email, profissão, faturamento, mensagem
- **E-books**: Inclui campo adicional para identificar qual e-book foi solicitado

## Personalização

Para personalizar:
1. **Formspree**: Substitua o endpoint por um formulário seu
2. **Links**: Atualize os links de redes sociais e botões de ação
3. **Conteúdo**: Edite textos, imagens e informações diretamente no HTML
4. **Estilos**: Modifique o CSS inline ou adicione novos estilos

## Suporte
